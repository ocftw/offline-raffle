# OCF 離線抽獎機

靜態網頁抽獎工具，無需網路、無需伺服器，下載或線上開啟即可使用。

## 功能

- 貼上名單，一鍵抽獎
- 輸入獎項名稱，大標題顯示於抽獎畫面
- 得獎紀錄自動累積，支援複製與清除
- 得獎紀錄暫存於瀏覽器 localStorage，重新整理後仍保留

## 使用方式

**線上版**：直接開啟 https://ocftw.github.io/offline-raffle

**離線版**：下載 `index.html`，用瀏覽器開啟即可，不需要網路。

## 注意事項

- 得獎紀錄只存在當下使用的瀏覽器，換裝置或無痕視窗不會保留
- 抽獎結束後請及時複製紀錄另存
